# ცილა-ცილოვანი ურთიერთქმედების ბმის აფინურობის პროგნოზირება

## აბსტრაქტი

ცილა-ცილოვანი ურთიერთქმედებები (PPI) ფუნდამენტურია თითქმის ყველა ბიოლოგიური პროცესისთვის და წარმოადგენს წამლების აღმოჩენის მთავარ ფოკუსს. ამ ურთიერთქმედებების ბმის აფინურობის ზუსტი პროგნოზირება გადამწყვეტია მოლეკულური ამოცნობის გასაგებად და თერაპიული ინტერვენციების შესამუშავებლად. მიუხედავად იმისა, რომ სტრუქტურაზე დაფუძნებული მეთოდები მაღალ სიზუსტეს გვთავაზობენ, ისინი ხშირად შეზღუდულია მაღალი გარჩევადობის 3D სტრუქტურების ხელმისაწვდომობით. ამ კვლევაში ჩვენ ვაფასებთ და ვალიდაციას ვუკეთებთ ISLAND (Interaction of Side-chains and Ligands with Atomic Network Descriptors) მეთოდოლოგიას მიმდევრობაზე დაფუძნებულ მიდგომას ვრცელ PPB-Affinity მონაცემთა ბაზაზე, რომელიც 12,000-ზე მეტ ცილოვან კომპლექსს შეიცავს. ჩვენ გამოვყავით 6,336 განზომილებიანი მახასიათებლების კომპლექსური ნაკრები, მათ შორის ამინომჟავური შემადგენლობა, ჩანაცვლების მატრიცის რეპრეზენტაციები და ფიზიკო-ქიმიური თვისებები. ჩვენმა შედეგებმა აჩვენა, რომ SVR-RBF მოდელი აღწევს პირსონის კორელაციას 0.508 ± 0.112, რაც მნიშვნელოვნად აღემატება ადრინდელ მიმდევრობაზე დაფუძნებულ სტანდარტებს. ეს ნაშრომი ადასტურებს მიმდევრობაზე დაფუძნებული დესკრიპტორების მდგრადობას ბმის აფინურობის დაფიქსირებაში და უზრუნველყოფს მასშტაბირებად ჩარჩოს დიდი მოცულობის PPI ანალიზისთვის სტრუქტურული მონაცემების არარსებობის შემთხვევაში.

უფრო დეტალური ანალიზისთვის, მათ შორის ცილების ბმის მნიშვნელობისა და ჩვენი მეთოდოლოგიის სრული აღწერისთვის, იხილეთ [THESIS.md](THESIS.md).
