# ცილებს შორის ბმის აფინურობის პროგნოზირება მიმდევრობაზე დაფუძნებული ISLAND მეთოდოლოგიით  
## PPB-Affinity მონაცემთა ბაზაზე

## აბსტრაქტი

### საფუძველი  
ცილებს შორის ბმის აფინურობის ზუსტი პროგნოზირება გადამწყვეტია ახალი მედიკამენტების აღმოჩენისა და ბიოლოგიური ურთიერთქმედებების უკეთ გასაგებად. მიუხედავად იმისა, რომ სტრუქტურაზე დაფუძნებული მეთოდები საჭიროებენ ცილების 3D სტრუქტურულ ინფორმაციას, მიმდევრობაზე (sequence) დაფუძნებული მიდგომები გამოირჩევა უფრო ფართო გამოყენებადობით. ამასთანავე, მათ წინაშე დგას პროგნოზირების სიზუსტესთან დაკავშირებული სირთულეები. ISLAND (Interaction of Side-chains and Ligands with Atomic Network Descriptors) მეთოდოლოგია წარმოადგენს პერსპექტიულ მიდგომას, რომელიც ბმის აფინურობის პროგნოზირებისთვის მხოლოდ ცილების მიმდევრობას იყენებს.

### მიზანი  
პროექტის მიზანია Abbasi-სა და სხვების (2020) მიერ შემოთავაზებული ISLAND მეთოდოლოგიის რეპროდუქცია და ვალიდაცია უფრო დიდ, დამოუკიდებელ მონაცემთა ბაზაზე — **PPB-Affinity**, რომელიც 2000-ზე მეტ ცილოვან კომპლექსს მოიცავს. კვლევის მიზანია მიმდევრობაზე დაფუძნებული მახასიათებლების განზოგადების უნარის შეფასება ცილებს შორის ბმის აფინურობის პროგნოზირების ამოცანაში და სხვადასხვა მანქანური სწავლების მიდგომების შედარება.

### მეთოდები  
ჩვენ განვახორციელეთ ISLAND მეთოდოლოგიის მახასიათებლების ექსტრაქციის სრული ციკლი. თითოეული ცილოვანი კომპლექსისთვის, როგორც ლიგანდისა, ასევე რეცეპტორისთვის, გამოთვლილი იქნა ოთხი კატეგორიის მახასიათებელი:

1. ამინომჟავური შემადგენლობა (AAC, 20D)  
2. BLOSUM-62 ჩანაცვლების მატრიცის რეპრეზენტაცია (SMR, 20D)  
3. ProPy-ის კომპლექსური ფიზიკო-ქიმიური მახასიათებლები (1537D)  
4. ProtParam-ის ბიოქიმიური თვისებები (7D)  

ლიგანდისა და რეცეპტორის მახასიათებლების, მათი აბსოლუტური სხვაობებისა და ნამრავლების კონკატენაციის შედეგად მივიღეთ **6,336-განზომილებიანი მახასიათებლების ვექტორები**. შეფასდა მანქანური სწავლების ოთხი მოდელი:

- Support Vector Regression RBF ბირთვით (SVR-RBF)  
- Linear SVR  
- Random Forest  
- Gradient Boosting  

გამოყენებულ იქნა **5-ჯერადი ჯგუფური კროს-ვალიდაცია (5-fold Group Cross-Validation)**, რათა თავიდან აგვეცილებინა მსგავსი სტრუქტურის მქონე ცილოვან კომპლექსებს შორის მონაცემთა გაჟონვა. მოდელების ეფექტურობა შეფასდა შემდეგი მეტრიკებით: პირსონის კორელაცია, სპირმენის კორელაცია, RMSE, MAE და \(R^2\).

### შედეგები  
ხარისხობრივი ფილტრაციის შემდეგ წარმატებით დამუშავდა **1,536 ცილოვანი კომპლექსი**. მიღებულმა შედეგებმა აჩვენა, რომ მხოლოდ მიმდევრობაზე დაფუძნებულ მახასიათებლებს შეუძლიათ ბმის აფინურობის ეფექტური პროგნოზირება. საუკეთესო შედეგი აჩვენა **SVR მოდელმა RBF ბირთვით**:

- **პირსონის კორელაცია:** 0.508 ± 0.112  
- **სპირმენის კორელაცია:** 0.440 ± 0.117  
- **RMSE:** 1.670 ± 0.154 pKd ერთეული  
- **MAE:** 1.303 ± 0.081 pKd ერთეული  
- **\(R^2\):** 0.239 ± 0.106  

მიღებული შედეგები წარმოადგენს მნიშვნელოვან გაუმჯობესებას ორიგინალური ISLAND სტატიის გარე ვალიდაციის მაჩვენებლებთან შედარებით (პირსონი: 0.35, RMSE: 1.98). Random Forest მოდელმა ასევე აჩვენა პერსპექტიული შედეგები, მაშინ როცა ხაზოვანმა მოდელებმა ვერ შეძლეს მონაცემებში არსებული რთული არახაზოვანი კავშირების სრულფასოვანი ასახვა.

### დასკვნა  
წარმოდგენილ ნაშრომში წარმატებით განხორციელდა ISLAND მეთოდოლოგიის ვალიდაცია დიდ, დამოუკიდებელ მონაცემთა ბაზაზე. გაუმჯობესებული შედეგები (პირსონის კორელაცია 0.508 vs 0.35–0.44) მიუთითებს, რომ:

1. მიმდევრობაზე დაფუძნებული მახასიათებლების ექსტრაქციის მიდგომა არის მყარი და განზოგადებადი  
2. უფრო დიდი მონაცემთა ბაზები ხელს უწყობს მოდელის ეფექტურობის ზრდას  
3. ბირთვზე დაფუძნებული მეთოდები (SVR-RBF) განსაკუთრებით ეფექტურია არახაზოვანი კავშირების დასაფიქსირებლად  

ამავდროულად, საშუალო კორელაციის მაჩვენებლები (r ≈ 0.5) ხაზს უსვამს მხოლოდ მიმდევრობაზე დაფუძნებული პროგნოზირების ფუნდამენტურ შეზღუდვებს, რადგან ამ მიდგომით ვერ ხდება ცილების სტრუქტურული დინამიკის სრული გათვალისწინება.
